## Reproducing

```bash
node dist/cli.js benchmark daily \
  --repo /path/to/repo \
  --task all \
  --mode baseline
# then
node dist/cli.js benchmark daily \
  --repo /path/to/repo \
  --task all \
  --mode compiled-packet
```

(The CLI does not accept comma-separated `--mode` values; run each mode separately or use `--mode all` for the full set.)
